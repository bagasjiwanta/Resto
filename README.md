<h1>Resto</h1>
Just an imaginary company for searching restaurants all across Brooklyn, Staten Island, Bronx, Manhattan, and much more

<h3>Built using :</h3>

1. ReactJs
2. Vanilla css
3. Mongodb

<h3>Features implemented :</h3>

1. Routing
2. Reusable components
3. Responsive design
4. API calls 
5. Deployment
And more :)
